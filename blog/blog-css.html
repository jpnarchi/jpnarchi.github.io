¿Qué es CSS?

CSS (Cascading Style Sheets) o Hojas de Estilo en Cascada es un lenguaje que se utiliza para describir la presentación de documentos HTML o XML, incluyendo colores, diseño, y fuentes. Permite separar el contenido de un documento web (HTML) de su presentación visual.

CSS fue desarrollado por el W3C (World Wide Web Consortium) y se ha convertido en un estándar fundamental para el diseño web moderno. Con CSS podemos controlar el aspecto visual de múltiples páginas web a la vez, lo que facilita enormemente el mantenimiento y la coherencia visual de un sitio web.

Componentes principales de CSS

Selectores

Los selectores son patrones que se utilizan para seleccionar los elementos HTML a los que queremos aplicar estilos. Indican a CSS qué elementos HTML deben ser afectados por los estilos declarados.

p { color: blue; } /* Selector de tipo: selecciona todos los párrafos */

Propiedades

Las propiedades son identificadores que indican qué característica de estilo queremos modificar. Ejemplos de propiedades incluyen color, font-size, margin, padding, etc.

h1 { font-size: 24px; } /* 'font-size' es la propiedad */

Valores

Los valores son asignaciones específicas para las propiedades. Por ejemplo, para la propiedad color, los valores pueden ser nombres de colores como red, códigos hexadecimales como #FF0000, o funciones como rgb(255, 0, 0).

p { color: #0000FF; } /* '#0000FF' es el valor para la propiedad 'color' */

Tipos de Selectores

Selector de Tipo

Selecciona todos los elementos que coinciden con el nombre del elemento especificado.

p {
  color: blue;
  font-size: 16px;
}

Este ejemplo seleccionará todos los elementos <p> en la página y aplicará los estilos definidos.

Selector de Clase

Selecciona elementos con un atributo de clase específico. Se denota con un punto (.) seguido del nombre de la clase.

.destacado {
  background-color: yellow;
  font-weight: bold;
}

Este ejemplo seleccionará cualquier elemento HTML que tenga el atributo class="destacado", como por ejemplo: <p class="destacado">.

Selector de ID

Selecciona un elemento con un atributo ID específico. Se denota con un hash (#) seguido del nombre del ID.

#cabecera {
  background-color: black;
  color: white;
}

Este ejemplo seleccionará el elemento HTML que tenga el atributo id="cabecera", como por ejemplo: <header id="cabecera">.

¡Importante! Los ID deben ser únicos en un documento HTML, mientras que las clases pueden aplicarse a múltiples elementos. Esta es la diferencia fundamental entre los selectores de ID y de clase.

Propiedades CSS Esenciales

A continuación se presentan algunas de las propiedades más utilizadas en CSS:

Propiedad Descripción Ejemplo
color Define el color del texto color: #333333;
background-color Define el color de fondo de un elemento background-color: #f8f8f8;
font-family Define la familia de fuentes a utilizar font-family: Arial, sans-serif;
font-size Define el tamaño de la fuente font-size: 16px;
margin Define el espacio alrededor de un elemento margin: 10px 20px;
padding Define el espacio entre el contenido y el borde padding: 15px;
border Define el borde alrededor de un elemento border: 1px solid #dddddd;
width Define el ancho de un elemento width: 100%;
height Define la altura de un elemento height: 50px;

Uso de CSS en un documento HTML

Existen tres formas principales de incluir CSS en un documento HTML:

1. CSS Interno (Internal CSS)

Se define dentro de la etiqueta <style> en la sección <head> del documento HTML.

<head>
  <style>
    body {
      font-family: Arial, sans-serif;
      margin: 20px;
    }
  </style>
</head>

2. CSS Externo (External CSS)

Se define en un archivo .css separado y se vincula al documento HTML mediante la etiqueta <link>.

<head>
  <link rel="stylesheet" href="estilos.css">
</head>

3. CSS en línea (Inline CSS)

Se aplica directamente a un elemento HTML mediante el atributo style.

<p style="color: blue; font-size: 16px;">Este es un párrafo con estilo en línea.</p>
